# CampusFlow

CampusFlow is a smart task reminder system that automates reminders using workflows.

## Live Demo

Frontend:
https://campusflow-aa5j.vercel.app/

Backend:
https://campusflow-hm4a.onrender.com

## Tech Stack
- React
- Node.js
- Express
- n8n
- Google Calendar

## Workflow
User Input → Frontend → Backend → n8n Webhook → Google Calendar Event
